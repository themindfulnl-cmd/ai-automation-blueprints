### 🤖 AI Agency Framework

Most people use one AI assistant for everything. That's like hiring one employee to do sales, analytics, marketing, and quality control simultaneously.

The **Agent Framework** is an open-source blueprint for building specialized AI agents — each with a narrow role, defined processes, and explicit anti-patterns (rules about what to NEVER do).

**Why narrow agents beat general prompts:**
*   **Identity** — Each agent has a distinct voice and expertise area
*   **Context** — Agents know the business data relevant to their role
*   **Process** — Step-by-step workflows, not vague instructions
*   **Anti-Patterns** — Explicit failure prevention eliminates hallucinated outputs

**Starter agents included:**

| Agent | Role | What It Does |
|---|---|---|
| Sales Agent | Outreach & Proposals | Cold DMs, follow-ups, partnership pitches — value-first, never spammy |
| Analytics Agent | Metrics & Reporting | Weekly reports, post scoring, growth tracking, content pillar analysis |
| Community Agent | Engagement & Relationships | Comment strategy (AVI framework), reply management, DM sequences |
| Brand Guardian | Quality & Consistency | Pre-publish review, vocabulary enforcement, cross-platform consistency |

Each example includes a complete SKILL.md template you can customize for your own business.
