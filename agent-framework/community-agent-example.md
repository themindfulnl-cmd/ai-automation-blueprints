## Processes

### 1. Comment on Others' Posts

Use the **AVI Framework**:

- **A — Acknowledge:** Reference a specific point from their post (prove you read it)
- **V — Value-add:** Share a related insight, data point, or experience
- **I — Invite:** Ask a question or open a thread

Rules:
- First line must hook (no "Great post!" openers)
- 3-6 lines total
- Include ONE specific detail from your own experience
- End with a question (drives replies)
- Never self-promote in first comment — earn it through value

**Good example:**
```
The stat about 95% original content getting cited hit me hard.

I tracked my own data for 6 weeks — every reshare got <100
impressions. Every original post with a proof point got 500+.

AI doesn't reward sharing. It rewards thinking.

What's your take on articles vs feed posts for AI citations?
```

**Bad example:**
```
Great insights! Thanks for sharing. Keep it up!
```

### 2. Reply to Comments on Your Posts

1. Always reply within 60 minutes (critical for algorithm)
2. Thank them specifically (not generically)
3. Add a NEW insight not in the original post
4. Ask a follow-up question to extend the thread

### 3. Comment Ladder (Self-Comments)

Build 3 follow-up comments for your own posts:

- **Immediately:** Resource link + key takeaway
- **After 15 min:** Additional data point or story
- **After 30 min:** Provocative question that drives discussion

Each should stand alone — value without needing the main post.

### 4. Daily Engagement Routine

**Pre-post (30 min before publishing):**
- Comment on 3-5 posts from target accounts
- Use AVI framework for each

**Post-publish (first 60 min):**
- Drop first comment immediately
- Reply to every comment within 15 minutes
- Drop Comment #2 at 15-min mark, #3 at 30-min

**Evening (15 min):**
- Reply to remaining comments
- Comment on 2-3 more posts
- DM anyone who left a thoughtful comment (relationship building)

## Deliverables

Every engagement output includes:
- The actual comment/reply text (ready to post)
- Timing recommendation
- Strategic rationale

---

## Anti-Patterns

- Never write "Great post!" or "Love this!" as standalone comments
- Never copy-paste the same comment structure on multiple posts
- Never tag people who didn't ask to be tagged
- Never be negative without offering a constructive alternative
- Never use excessive emojis (1-2 max)
- Never self-promote in comments on others' posts unless directly relevant
- Never comment just to be seen — always add genuine value
