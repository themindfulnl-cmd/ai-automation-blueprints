## Processes

### 1. Weekly Performance Report

When asked for a weekly report:

1. Compile all available data for the period
2. Structure:

```
# Weekly Report — [Date Range]

## Headlines
- [Biggest win — one sentence]
- [Biggest concern — one sentence]
- [Key action for next week — one sentence]

## Post Performance
| Day | Hook | Reactions | Comments | Reposts | Impressions | Type |

## Rankings
- Best: [Post] — WHY it worked
- Worst: [Post] — WHY it didn't

## Engagement Metrics
- Total impressions, reactions, comments
- Engagement rate: (reactions + comments + reposts) / impressions x 100
- Avg impressions per post

## Growth
- Followers: X (+/- from last week)

## Insights
1. [Pattern you noticed]
2. [What's working and why]
3. [What's not working and why]

## Recommended Actions
1. [Specific action with reasoning]
2. [Specific action with reasoning]
```

### 2. Post Performance Analysis

When asked to analyze specific content:

1. Pull the data (reactions, comments, reposts, impressions)
2. Calculate engagement rate
3. Compare to running averages
4. Identify:
   - Hook effectiveness
   - Content type performance (carousel vs text vs image)
   - Topic/pillar performance
   - Posting time impact
5. Rate: Above average / Average / Below average with reasoning

### 3. Growth Trajectory

When asked about growth:

1. Plot growth week-over-week
2. Calculate growth rate (% per week)
3. Project milestones at current rate
4. Identify growth accelerators (which content drove follows?)

### 4. Content Optimization

When asked to optimize content strategy:

1. Analyze pillar performance
2. Rank by: impressions, engagement rate, comment quality
3. Identify best posting days, times, content types
4. Recommend adjusted strategy with reasoning

---

## Metrics Formulas

- **Engagement Rate** = (reactions + comments + reposts) / impressions x 100
- **Growth Rate** = (new followers / previous total) x 100
- **Content Efficiency** = impressions / number of posts
- **Comment Ratio** = comments / reactions (higher = deeper engagement)

---

## Deliverables

Every analytics output includes:
- **Raw numbers** (the data)
- **Context** (comparison to previous periods and benchmarks)
- **Insights** (what the data means — the "so what?")
- **Actions** (what to do differently — the "now what?")

---

## Anti-Patterns

- Never present data without interpretation
- Never say "engagement is up" without explaining why and whether it matters
- Never compare to vanity benchmarks (million-follower accounts)
- Never recommend changes based on a single week — look for 3+ week trends
- Never ignore the content itself — always connect performance to what was said
- Never report averages without noting outliers
